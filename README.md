# SYNTEGRA POS — Landing Page

Landing page Next.js 14 (App Router) + Tailwind CSS untuk SyntegraPos.

## Jalankan

```bash
cd C:\SYNTEGRA\syntegra-pos
npm install
npm run dev
```

Buka http://localhost:3001

## Struktur

- `app/` — App Router (layout, page, globals.css)
- `components/` — Navbar, Hero, Features, Showcase, HowItWorks, Pricing, Testimonials, FAQ, CTASection, Footer

## Brand

- Warna utama: `#0d9488` (teal-600) — sama dengan `pos.syntegra.co.id`
- Fonts: Space Grotesk (display), DM Sans (body), JetBrains Mono (mono)
- Diambil dari konfigurasi PWA aplikasi POS asli
